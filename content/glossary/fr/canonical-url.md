## Définition

Une **URL canonique** est l’URL unique qu’un éditeur désigne comme version faisant autorité d’une page. Elle est déclarée via `<link rel="canonical" href="..." />` dans le `<head>` HTML (ou en-têtes HTTP). Les crawlers et agents l’utilisent pour éviter la confusion de contenu dupliqué.
