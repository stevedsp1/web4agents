## Définition

Les **breadcrumbs** (fil d’Ariane) sont la piste de navigation (ex. Accueil > Docs > Schema.org) qui indique où se situe la page dans la hiérarchie du site. Pour les agents, ils doivent être accompagnés du JSON-LD Schema.org **BreadcrumbList** pour donner un signal de hiérarchie explicite.
