## Définition

**OpenAPI** (ex-Swagger) est un format pour décrire les endpoints, paramètres et schémas d’une API. Les agents IA peuvent utiliser cette spécification pour découvrir et appeler votre API de façon autonome. Une API bien documentée en OpenAPI est un atout pour la GEO.
