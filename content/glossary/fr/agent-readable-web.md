## Définition

L’**agent-readable web** (web lisible par les agents) est l’idée d’un web où le contenu et la structure sont conçus pour être consommés par les agents IA autonomes : données structurées, sémantique claire, actions identifiables et limites explicites (robots.txt, Content Signals).
