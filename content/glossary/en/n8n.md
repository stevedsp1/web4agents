## Overview

n8n is a fair-code workflow automation tool that enables agents and humans to orchestrate tasks across hundreds of integrations.

## Use cases

- Automating data pipelines for GEO audits
- Connecting CMS and structured data sources
- Triggering actions from agent decisions
