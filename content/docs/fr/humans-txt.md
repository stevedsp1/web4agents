## Qu’est-ce que humans.txt ?

Le fichier `humans.txt` est une convention volontaire : un fichier texte à la racine du site (`https://example.com/humans.txt`) qui décrit l’équipe, les technologies et les objectifs du site — pour les humains qui regardent le code source.

## Les agents IA l’utilisent-ils ?

Certains crawlers et outils peuvent le lire pour des signaux E-E-A-T (qui est derrière le site). Ce n’est pas un standard pour les IA comme l’est `llms.txt`. Pour les agents IA, privilégiez **llms.txt** pour le contexte et la structure ; `humans.txt` reste un complément utile pour la confiance et la transparence.

## Comment le créer

Contenu en texte libre : équipe, technologies, contact. Pas de format imposé. Placez le fichier à la racine et, si vous le souhaitez, ajoutez un lien dans le pied de page (« Créé par… »). Voir le [guide E-E-A-T](/fr/docs/eeat) pour l’usage des signaux de confiance.
