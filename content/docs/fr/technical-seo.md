## Pourquoi le SEO technique compte encore pour les agents

Les agents IA et les moteurs de recherche ont besoin d’URLs stables, de métadonnées cohérentes et d’un site crawlable. Le SEO technique pose les bases ; la GEO s’appuie dessus.

## Points clés

- **URLs canoniques** : Une URL canonique par ressource pour éviter le contenu dupliqué.
- **Balises meta** : `title`, `description`, Open Graph pour chaque page importante.
- **Hreflang** : Pour les sites multilingues, indiquer les variantes de langue pour une attribution correcte.
- **Pagination** : `rel="next"` / `rel="prev"` ou sitemap pour les listes paginées.
- **Codes HTTP** : 200 pour le contenu valide, 301/302 pour les redirections, 404 pour l’absent.

Les crawlers IA utilisent les mêmes signaux que les moteurs. Voir [Internal linking](/fr/docs/internal-linking), [Content freshness](/fr/docs/content-freshness), [Backlinks](/fr/docs/backlinks) et [E-E-A-T](/fr/docs/eeat).
